# Ad Audit & Optimization System

## Overview

This skill provides comprehensive advertising audit and optimization capabilities based on 42 proven D2C performance marketing frameworks. It analyzes ads across multiple dimensions—creative effectiveness, profitability alignment, technical setup, and scalability—then provides detailed technical recommendations and optimized variations.

## Core Capabilities

### 1. Ad Auditing
- Creative conversion framework analysis (PAS, BAB, Myth Breaker, Social Proof, Future State formulas)
- Hook effectiveness evaluation against proven triggers
- Offer engineering assessment (value stack, risk reversal, urgency)
- Ad fatigue detection and diagnosis
- Cost efficiency analysis per creative
- Performance metrics alignment (CTR, CVR, MER, ROAS, CM2, EBITA)

### 2. Campaign Analysis
- Google Ads account structure audit (Search, Shopping, PMax, YouTube)
- Meta campaign performance evaluation
- Attribution clarity assessment
- Tracking accuracy verification
- Budget distribution optimization
- Profitability layer analysis (CM1, CM2, CM3, EBITA)

### 3. Copy Optimization
- Rewrite ad copy using conversion frameworks
- Generate A/B test variations with single-variable testing
- Apply hook testing principles from swipefile
- Optimize for specific emotional triggers (curiosity, contrarian, emotional, logical, outcome, social proof, urgency)
- Ensure brand voice consistency while applying frameworks

### 4. Problem Identification & Solutions
- Detect profit leaks across acquisition, fulfillment, retention
- Identify scaling mistakes (tracking issues, ROAS vs MER confusion, premature channel expansion, margin blindness, creative burnout, cash flow gaps)
- Diagnose ad fatigue sources (creative vs offer vs funnel)
- Find hidden inefficiencies in campaign structure

## Audit Process

### Step 1: Data Collection
Gather all available information:
- Ad copy (headlines, body text, CTAs)
- Creative assets (images, video descriptions, visual elements)
- Performance data (CTR, CVR, CPC, CPA, ROAS, MER, spend, revenue)
- Campaign structure details
- Landing page URLs (if provided)
- Profitability metrics (COGS, CM1, CM2, LTV, CAC if available)

### Step 2: Framework Application
Apply relevant frameworks from references:
- **Creative Analysis**: Use Creative Conversion Framework, Hook Testing Swipefile
- **Profitability**: Apply MER vs ROAS analysis, CM2 calculation, EBITA alignment
- **Technical Setup**: Reference Google Ads Audit Kit, Tracking Survival Kit
- **Scaling Readiness**: Use Precision Scaling Checklist, 7 Deadly Scaling Mistakes
- **Offer Assessment**: Apply Offer Engineering Framework

### Step 3: Comprehensive Audit Report
Generate formal, technical audit covering:

**Executive Summary**
- Overall performance assessment
- Critical issues identified (prioritized by profit impact)
- Quick wins available
- Long-term optimization opportunities

**Creative Effectiveness Analysis**
- Storytelling structure evaluation
- Hook strength assessment (first 3 seconds/opening line)
- Emotional trigger identification and effectiveness
- Visual/message alignment
- Format optimization (UGC vs motion vs static)
- Creative lifespan and fatigue indicators

**Performance Metrics Assessment**
- CTR analysis and benchmarking
- CVR evaluation and funnel alignment
- Cost efficiency calculation (CE = (Revenue - Ad Spend) / Ad Spend)
- MER analysis (total revenue / total ad spend)
- ROAS vs MER discrepancy identification
- CM2 impact evaluation
- LTV:CAC ratio assessment (if data available)

**Campaign Structure Audit**
- Account organization review
- Segmentation effectiveness (branded vs non-branded, cold vs warm)
- Budget distribution analysis
- Bid strategy alignment
- Tracking accuracy verification
- Attribution model assessment

**Profitability Alignment**
- Contribution margin analysis (CM1, CM2, CM3)
- Break-even ROAS calculation
- Profit leak identification
- CAC efficiency relative to LTV
- Cash flow impact assessment
- Scaling safety threshold evaluation

**Offer Engineering Evaluation**
- Value stack assessment
- Risk reversal presence and strength
- Urgency implementation (ethical vs fake scarcity)
- Offer presentation clarity
- Conversion friction points

### Step 4: Optimization Deliverables

**Rewritten Ad Variations**
Provide 3-5 optimized versions using different frameworks:
- Version 1: PAS (Problem-Agitation-Solution)
- Version 2: BAB (Before-After-Bridge)  
- Version 3: Social Proof Loop
- Version 4: Contrarian/Myth Breaker
- Version 5: Future State Formula

Each version includes:
- Optimized headline (applying hook principles)
- Body copy (structured by chosen framework)
- Clear CTA
- Rationale for framework selection
- Expected performance improvements

**A/B Test Recommendations**
- Single-variable test suggestions (hook only, offer only, visual format only)
- Testing timeline and budget allocation
- Success criteria and measurement framework
- Statistical significance guidelines

**Technical Fixes**
- Tracking corrections needed
- Campaign structure improvements
- Budget reallocation recommendations
- Bid strategy adjustments
- Negative keyword additions
- Feed optimization requirements

**Strategic Recommendations**
- Scaling readiness assessment
- Channel expansion timing
- Creative refresh cadence
- Retention integration opportunities
- Profitability protection measures

When user asks about setup/strategy:
1. Read relevant references (google-ads-optimization.md, scaling-principles.md)
2. Reference appropriate audit framework
3. Apply profitability lens
4. Check against scaling readiness criteria
5. Provide technical recommendations
6. Include implementation timeline

## Critical Principles

**Always Maintain**:
- Formal, technical tone
- Data-driven recommendations
- Profit-first perspective (MER and CM2 over ROAS alone)
- Framework attribution (cite which framework informs each recommendation)
- Prioritization by financial impact

**Never**:
- Recommend discounting as primary strategy
- Ignore contribution margins in scaling decisions
- Suggest creative changes without framework rationale
- Mix multiple emotional triggers in single creative
- Recommend scaling without tracking verification
